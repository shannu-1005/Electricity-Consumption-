# Plugging into the Future: An Exploration of Electricity Consumption Patterns

# Project Description

Plugging into the Future: An Exploration of Electricity Consumption Analysis is a Tableau-based data analytics project that examines electricity consumption patterns across different Indian states using historical data. It visualizes trends, seasonal variations, and the impact of COVID-19 on power usage through interactive dashboards. The project helps identify consumption patterns and supports data-driven energy planning and decision-making.
